State Space

$$ x = \begin{bmatrix} p \\ \dot{p} \end{bmatrix} $$ state = [ position, velocity ]
$$ u = \begin{bmatrix} f \end{bmatrix} $$ input = [ force ]
$$ \dot{x} = \mathbf{A}x + \mathbf{B}u $$ state_delta = A * state + B * input
